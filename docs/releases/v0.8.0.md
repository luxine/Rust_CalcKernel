# v0.8.0 Historical Note

This file is kept as migrated historical context from the TypeScript-era
CalcKernel documentation. The current Rust repository ships `native ckc`; use
`docs/native-release.md` and `docs/RELEASE_CHECKLIST.md` for current release
instructions.

The relevant lasting contract from this release line is strict `f64` support:

- `f64` is the only CK floating point type.
- implicit integer / floating point conversion is not supported.
- exact explicit casts are exposed through `i32_to_f64` and `u32_to_f64`.
- generated C, WASM, and LLVM artifacts preserve the CK type contract.

For the current Rust-native product boundary, see `README.md`.
